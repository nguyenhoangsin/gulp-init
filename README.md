To install Yarn with npm, run:
```bash
npm install -g yarn
```
Move to the [project] directory, run:
```bash
cd [project]
```

### Install dependencies
With npm, run:
```bash
npm install
```
With Yarn, run:
```bash
yarn
```

### Run project
Development environment, run:
```bash
npm run dev
```

### Build project
Build production, run:
```bash
npm run build
```
